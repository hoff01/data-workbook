Open PowerShell in this folder and run:

powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\scripts\install_windows_task.ps1

That installs Python requirements and creates a Windows Task Scheduler task named:

EIA Summary Dashboard

Schedule:

Wednesday at 9:30 AM

To add recipients, edit email_recipients.txt and put one email address per line.

The task downloads the EIA historical .xls workbooks and WPSR CSV first, rebuilds raw.csv.tar.xz and series.csv inside this folder, then builds and sends the dashboard PDF.

To run once immediately:

powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\scripts\run_scheduled_email.ps1
